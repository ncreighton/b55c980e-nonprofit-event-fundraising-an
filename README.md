Integrates with Stripe, PayPal, and Mailchimp. Built for foundations, charities, and community organizations needing a turnkey event and donor management system without expensive CRM licenses.

== Description ==

A comprehensive WordPress plugin for nonprofits that manages fundraising events, volunteer sign-ups, and donor acknowledgment workflows. Features ticketed event pages, recurring donation support, volunteer shift scheduling, automated tax receipt generation, and CRM-style donor history tracking. Integrates with Stripe, PayPal, and Mailchimp. Built for foundations, charities, and community organizations needing a turnkey event and donor management system without expensive CRM licenses.

**Features:**

* Core functionality

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate via WordPress admin > Plugins
3. Configure via Settings > Nonprofit Event Fundraising and Volunteer Management WordPress Plugin
